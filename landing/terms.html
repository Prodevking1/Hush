3. Description du service

Hush est une application macOS qui corrige automatiquement le texte que vous tapez dans n'importe quel champ de texte de votre Mac. L'application fonctionne comme suit :

  1. Détection de la pause : Hush détecte lorsque vous cessez de taper (délai configurable).
  2. Lecture du texte : Le contenu du champ de texte actif est lu via l'Accessibility API de macOS.
  3. Correction par IA : Le texte est envoyé à un modèle d'intelligence artificielle (Ministral 3B de Mistral AI, via OpenRouter) pour correction.
  4. Remplacement : Le texte corrigé remplace automatiquement l'original dans le champ.

L'application propose plusieurs modes : correction orthographique et grammaticale, reformulation, mode personnalisé. La détection de la langue est automatique (français, anglais et autres langues supportées).

  • Connexion internet (pour le mode cloud) ou aucune connexion (futur mode local)
  • Une clé API OpenRouter (fournie par l'utilisateur)
